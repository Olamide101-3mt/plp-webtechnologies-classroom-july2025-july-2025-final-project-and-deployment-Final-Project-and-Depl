About AuroraShop

AuroraShop is a small, conceptual e-commerce demo built to showcase responsive design, front-end interactivity, and clean code organization.

Features include product listings, a localStorage-backed cart, responsive navigation, and simple checkout simulation. This project was built for a Web Development portfolio and as a submission for a PLP assignment.
